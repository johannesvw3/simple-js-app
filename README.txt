# Pokedex app
Building a web app with **HTML**, **CSS** and **JS** that loads data from an external API(https://pokeapi.co/api/v2/pokemon).

### Function
The users will be able to click through a list of pokemons and see more details on demand.

### Features
* Load data from an external source (API).
* Display items on a list

### Built with

* JavaScript
* jQuery
* HTML
* CSS
* Bootstrap
